## 六、标题

### 语法：

## 八、文字显示

### 1、字体

语法：

- 粗体：用一对双星号包裹
- 删除线：用一对双飘号包裹
- 下划线：用一对u标签包裹
- 斜体：用一对单星号包裹
- 高亮：用一对双等号包裹

代码:

```
**这是粗体**
~~这是删除线~~
<u>这是下划线</u>
*这是斜体*
==这是高亮==
```

效果：

**这是粗体**
~~这是删除线~~
<u>这是下划线</u>
*这是斜体*
==这是高亮==

快捷键：

- 加粗：ctrl+B
- 删除线：shift+alt+5
- 下划线：ctrl+u
- 斜体：ctrl+i



如果想输入乘号但是发生斜体   如：123*123*123

就需要转义字符123\*123\*123

```
123\*123\*123
```
